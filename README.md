# Kanso
*A Japanese Zen concept meaning “simplicity” or “elimination of clutter.”*

Kanso is a minimalist writing app designed for distraction-free note-taking and journaling. It features theme selection, font customization, word/paragraph/page limit controls, and file operations, all stored locally for privacy and speed.



## Features
- Minimalist, clean interface
- Light, Coffee, and Dark themes
- Font family selection (Georgia, Merriweather, Lora, Open Sans, Playfair, Courier Prime, etc.)
- Word, paragraph, and page limits with warnings
- Live stats panel (characters, words, paragraphs, pages)
- Save, open, and create new documents (local file system)
- Persistent settings and document via localStorage
- Remembers user preferences for modes (theme, font, limits, etc.)
- Responsive design for desktop and mobile


## Usage
1. Start writing in the main text area.
2. Use the controls panel to change theme, font, or set word/paragraph/page limits.
3. Save your work to your device, open existing files, or start a new document.
4. All preferences are remembered between sessions.


## Keyboard Shortcuts
- **Ctrl+S / Cmd+S**: Save document
- **Ctrl+O / Cmd+O**: Open document
- **Ctrl+N / Cmd+N**: New document
- **Esc**: Close modal dialogs


## Technologies
- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (vanilla)
- Vite (development server & build)
- Tailwind CSS (utility-first styling)
- Font Awesome for icons
